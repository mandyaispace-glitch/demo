# 精力管理評測系統：未來資安防護與整體成本評估報告

本報告針對系統未來的**資訊安全 (Infosec) 潛在疑問**進行深入剖析，並提供結合資安防護後的**整體維運成本分析**，協助您在系統擴大規模時，做出最合適的技術決策。

---

## 🔒 一、 未來潛在的資安疑問與防禦對策

隨著您的客戶數量增加（從 5 個增加到 50 個以上），系統可能會面臨以下資安挑戰，本架構已預留了相對應的防禦空間：

### 1. 台灣《個資法》合規性與責任歸屬
*   **資安疑問**：系統收集學員的「姓名、Email、精力評測分數」，一旦外洩，是否有個資法法律責任？
*   **架構防禦**：
    *   由於我們採用**「一客一庫、獨立 Google Sheet」**設計，每個客戶（企業/獨立顧問）都是使用**他們自己的 Google 帳號**來儲存學員資料。
    *   在法律責任上，您是「資料處理者（Data Processor）」，而客戶才是「資料控制者（Data Controller）」。資料安全性由 Google 伺服器防護，責任歸屬於客戶的 Google 帳號權限管理。
    *   這大幅降低了您作為平台方所承擔的集中式洩密風險。

### 2. Token 暴力破解風險 (Brute Force)
*   **資安疑問**：如果某人知道學員的 Email，是否可以透過程式不斷測試 Token (例如 `a8f9`) 來竊取報告？
*   **架構防禦**：
    *   **長金鑰升級**：目前測試使用的是 4 碼 Token。未來正式上線時，我們可以將 Token 升級為 **16 碼隨機字元**（例如 `x8f9-2a8d-9c3f`）或 **UUID**。這在數學上是絕對無法被暴力破解的。

### 3. Google Sheet 遭內部人員誤刪或竄改
*   **資安疑問**：客戶的顧問或助教在操作試算表時，如果不小心刪除公式、整列數據或竄改學員分數該怎麼辦？
*   **架構防禦**：
    *   **欄位保護功能**：可在 Google Sheet 中設定「保護工作表與範圍」，將前測/後測的分數欄位（E 欄到 AZ 欄）設為**「僅限 API 帳號寫入，其餘人唯讀」**。顧問只能編輯姓名、Email 與 Token。
    *   **版本歷史回溯**：Google Drive 內建強大的版本歷程記錄，若遭誤刪，可一鍵還原到過去任何一分鐘的狀態。

## 💰 二、 整體維運成本分析（包含資安防護）

我們將維運規模分為三個階段，評估在不同階段下，您所需要支付的雲端與資安防護成本：

### 階段一：初創期（5 - 10 個客戶）── 極致省錢
此階段以快速驗證、低成本運作為主。
*   **伺服器/網頁託管 (Vercel)**：**$0 / 月** (使用 Hobby 免費版，已內建免費 SSL 憑證加密 HTTPS)。
*   **資料庫 (Google Sheets & Apps Script)**：**$0 / 月** (完全免費)。
*   **資安防護成本**：**$0 / 月**。
    *   *防護動作*：要求所有合作顧問的 Google 帳號必須開啟「雙重驗證 (2FA)」。
*   **【整體每月花費：NT$ 0】**

---

### 階段二：成長期（10 - 30 個客戶）── 商業合規與穩定
此階段客戶增加，為符合商業使用規範並提升穩定度，建議升級部分服務。
*   **網頁託管 (Vercel Pro)**：**$20 / 月 (約 NT$ 650)**。
    *   *提升資安*：提供更強的 DDoS 防禦機制、網頁防火牆（WAF）以及 99.9% 系統在線保證。可託管多達 100 個獨立客戶網站。
*   **網網域年費 (Domain)**：每個客戶若需綁定獨立網網域，年費約 **NT$ 400 - 600 / 年**（通常可由客戶自行支付）。
*   **資料備份機制**：**$0 / 月** (使用 Apps Script 撰寫排程，每天半夜自動將試算表備份存檔至另一個備份雲端硬碟)。
*   **【整體每月花費：約 NT$ 650】**

---

### 階段三：企業期（50 個客戶以上）── 企業級安全審查
若有大型企業客戶（如台積電、聯發科等）購買您的服務，他們會要求進行嚴格的資安審查（如滲透測試、ISO 27001）。
*   **資料庫遷移 (至 Supabase 或 AWS)**：約 **$25 / 月 (約 NT$ 800)**。
    *   *原因*：企業客戶可能不接受資料存在 Google Sheet，此時需將資料庫遷移至專門的 PostgreSQL 加密資料庫，並設定嚴格的 IP 白名單。
*   **網頁託管 (Vercel Pro)**：**$20 / 月**。
*   **資安防護與備份**：約 **$15 / 月**。
*   **【整體每月花費：約 NT$ 1,800 - 2,500】**

在您目前 **5-6 個客戶** 的規模下：
1.  **實質資安成本為 $0**。只要做好 Google 帳號的密碼防護與 2FA 雙重驗證，Google 伺服器等級的防火牆已經幫您擋掉了 99.9% 的網路黑客。
2.  藉由**【方案 A: Magic Link 專屬連結】**與 **【16碼長 Token 驗證】**，您的學員個資與評測結果是高度安全的，完全具備商業化提案的資安底氣。
